How a text message becomes action

1. Send a plain text

Message Super like you would a person: “open Twitter and draft a reply,” “run my marketing play,” or “check prices and report back.” No app switching.

2. Super operates a computer

Behind the scenes, Super clicks, types, scrolls, and navigates real apps and browsers in the cloud—similar to how you would do it yourself.

3. Cache the workflow

Successful actions are saved to a computer-use cache, so repeated requests get faster, more reliable, and cheaper over time.

4. Follow-ups by text

Super replies in the same thread with results, previews, or questions when confirmation is needed.
